# Spring Boot API Fundamentals Course

## Repository for Spring Boot Course - SST-A

This repository contains all the course materials, assignments, homework, and code examples for the Spring Boot API Fundamentals course.

## Course Overview

**Duration:** 12 Weeks (24 Classes Ã— 2 Hours Each)  
**Total Lessons:** 24

This comprehensive course covers Spring Boot from fundamentals to advanced API development, including production deployment and portfolio building.

## Repository Structure

Each lesson folder contains the following subdirectories:

- **`study-materials/`** - Lecture notes, presentations, and reference materials
- **`assignments/`** - In-class assignments and exercises
- **`homework/`** - Homework assignments and projects
- **`code/`** - Code examples and implementations from the class

## Prerequisites

- Basic Java knowledge (OOP concepts, collections, exceptions)
- Understanding of HTTP and REST concepts
- Basic database knowledge (SQL)
- Familiarity with Maven or Gradle (preferred)

## Technology Stack

- **Backend:** Spring Boot 3.x, Spring Security, Spring Data JPA
- **Database:** MySQL/PostgreSQL, Redis (caching)
- **Documentation:** Swagger/OpenAPI 3
- **Testing:** JUnit 5, Mockito, TestContainers
- **Deployment:** Docker, AWS/Azure/GCP
- **Tools:** IntelliJ IDEA, Postman, Git
